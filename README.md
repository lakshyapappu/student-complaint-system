# Student Complaint Management System

This project is a beginner-friendly full-stack web application built with:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB with Mongoose

## Run the project

1. Open a terminal in `backend/`
2. Install dependencies:
   `npm install`
3. Make sure MongoDB is running locally on:
   `mongodb://127.0.0.1:27017/student_complaint_system`
4. Start the server:
   `npm start`
5. Open:
   `http://localhost:5000`

## Default flow

- Students can sign up, log in, register complaints, vote, see complaint history, and withdraw complaints.
- Admins can sign up as `admin`, log in, see complaint statistics, and update complaint status and remarks.
